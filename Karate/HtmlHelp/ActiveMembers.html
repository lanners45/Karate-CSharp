Active Members

The active members section allows you to maintain the active members of the group. The options avilable are

Add a member
Edit the selected member
Manage Documents. This option allows you to add new documents both start forms and other documents to the selected member and view existing documents.
Gradings Management. This option allows the user to Add new and delete existing gradings for the selected member.
Badges Management. This option allows the user to Add new and delete existing badges for the selected member.
Summary. This option allows he user to view the summary for the selected member.
Deactivate. This option allows he user deactive the selected member. The member will move to the deactive view where they can be reactivated.
Print. This option allows he user to the list of active members. The list will include Name, age, start date and class
